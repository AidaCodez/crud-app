## 📖 Overview

This project is a simple CRUD (Create, Read, Update, Delete) application built to demonstrate how the **PERN stack**—PostgreSQL, Express.js, React.js, and Node.js—can be used to create a full-stack web application. It provides a minimal, beginner-friendly example of how the frontend and backend interact with a database using RESTful APIs.

## 🚀 Features

- **Create** – Add new records to the PostgreSQL database
- **Read** – Retrieve and display existing data
- **Update** – Edit and update records in the database
- **Delete** – Remove records from the database

## 🛠️ Tech Stack

- **Frontend:** React, JavaScript, HTML, CSS
- **Backend:** Node.js, Express
- **Database:** PostgreSQL
- **Other Tools:** Postman, Bootstrap, etc.

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/AidaCodez/crud-app.git
cd crud-app
```

## ▶️ Running the Project

### Start backend

```
cd server
node index
```

### Start frontend

```
cd client
npm start
```
